# ProBE4T
High-Resolution Geological Time Modelling from Lithotype-Informed Sedimentation Rates

Data in text and  MATLAB mat-format.

DataFileF.txt: XRF data from the Faageltofta-2 well
meanRatesF.mat: Mean sedimentation rates estimated by the AstroComb program
varRatesF.mat: Variances of the sedimentation rates estimated by the AstroComb program
timesF.mat: Geological Time-Depth relation estimated by the AstroComb program

AstroComb is a probabilistic Milankovic analyser program available at: https://doi.org/10.5281/zenodo.17966228
